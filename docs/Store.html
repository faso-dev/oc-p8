find((query), (callback))

Source:

Trouve les éléments basés sur une requête donnée en tant qu'objet JS
Example
db.find({foo: 'bar', hello: 'world'}, function (data) {
	 données retournera tous les éléments qui ont foo: bar et
	 hello: world dans leurs propriétés
});
Parameters:
Name Type Description
(query) object La requête à comparer (c'est-à-dire {foo: 'bar'})
(callback) function La fonction de rappel à déclencher lorsque l' exécution de la requête est terminée.
